# FileShare Pro - Профессиональное приложение для совместного редактирования файлов

Приложение позволяет пользователям подключаться друг к другу для чтения и редактирования файлов с модальным окном запроса разрешения.

## Возможности

- ✅ **Подключение между пользователями** - режим сервера (хост) и клиента
- ✅ **Модальное окно разрешений** - запрос на чтение/редактирование при подключении
- ✅ **Настройки портов** - настройка портов сервера и клиента
- ✅ **Светлая и темная тема** - профессиональный интерфейс в стиле Windows 11
- ✅ **Редактор файлов** - просмотр и редактирование текстовых файлов
- ✅ **Общая папка** - выбор папки для предоставления доступа

## Технологии

- **PyQt6** - профессиональная графическая библиотека
- **Сокеты** - сетевое взаимодействие между клиентами
- **Потоки** - асинхронная обработка соединений

## Установка зависимостей

```bash
pip install PyQt6
```

## Запуск приложения

```bash
python file_share_pro.py
```

## Использование

### Режим сервера (хост)

1. Нажмите **"▶️ Запустить"** в секции "Сервер"
2. Укажите порт (по умолчанию 8080)
3. Выберите общую папку для доступа
4. При подключении клиента появится модальное окно запроса разрешения

### Режим клиента

1. Введите адрес хоста и порт
2. Нажмите **"🔗 Подключиться"**
3. Выберите запрашиваемые разрешения (чтение/редактирование)
4. После одобрения сервером получите доступ к файлам

### Настройки

Меню **Настройки → Настройки приложения**:
- Выбор темы (Светлая/Темная/Системная)
- Порт сервера (1024-65535)
- Порт клиента (1024-65535)
- Автоматический запуск сервера

## Структура приложения

```
file_share_pro.py
├── ThemeManager          # Управление темами (светлая/темная)
├── PermissionDialog      # Модальное окно запроса разрешений
├── FileEditorDialog      # Диалог редактора файлов
├── SettingsDialog        # Диалог настроек приложения
├── ServerThread          # Поток сервера для предоставления доступа
├── ClientThread          # Поток клиента для подключения
└── MainWindow            # Главное окно приложения
```

## Скриншоты интерфейса

Интерфейс включает:
- Панель инструментов с быстрым доступом
- Левая панель: управление сервером, клиентом, выбор папки
- Правая панель: таблица файлов с действиями
- Статус бар с информацией о состоянии

## Безопасность

- Запрос разрешения при каждом подключении
- Раздельные права на чтение и запись
- Предупреждение при выдаче прав на редактирование
